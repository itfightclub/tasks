## File Header

There here are some examples, however, use comments for the language being used.

```go
/*
SPDX-License-Identifier: MIT

<path>/<to>/<file>.go
*/
```

```sh
# SPDX-License-Identifier: MIT
# SPDX-FileCopyrightText: C 2026 https://github.com/itfightclub
# <path>/<to>/<file>.sh
```
